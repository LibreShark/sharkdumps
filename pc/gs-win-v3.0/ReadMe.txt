Last minute Changes and Trouble-shooting
----------------------------------------

If the video dosen't play when you launch GameShark for Windows
then it is possible that your setting's on Internet Explorer need
changing. Do the following:
		1. Launch Internet Explorer
		2. Goto Tools on the Menu
		3. Goto	Internet options
		4. Choose advanced.
		5. Check that 'Play Videos' and 'Play Sounds' are on.
